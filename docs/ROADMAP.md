# Roadmap

## Phase 1: MVP
- PWA shell + authentication
- CSV import with mapping and preview
- Ledger + review queue
- Categorization rules and transfer matching
- Dashboard summary + monthly reports
- Assets with manual valuations

## Phase 2: Quality
- PDF parsing + OCR
- Improved duplicate detection
- Category suggestions from history
- Data export improvements

## Phase 3: Automation
- Provider integrations (where available)
- Automated valuations for selected assets
- Scheduled monthly snapshot and reminders
