# Information Architecture

## Primary Navigation
- Dashboard
- Ledger
- Review Queue
- Assets
- Import Hub
- Monthly Close
- Settings

## Core Workflows
1) Import: Upload -> Map Columns -> Preview -> Detect Duplicates -> Complete.
2) Review: Unknowns -> Assign Categories -> Transfer Pairing -> Resolve.
3) Monthly Close: Summary -> Validate -> Export -> Snapshot.
4) Assets: Update Values -> Track History -> View Net Worth.

## Key Views
- Dashboard: net worth hero, spend breakdown, trend line, waterfall.
- Ledger: transaction list with filters, bulk actions, and detail drawer.
- Review Queue: cards for unknowns, transfers, and duplicates.
- Assets: asset class cards with valuation timeline.
- Monthly Close: monthly breakdowns and exports.
